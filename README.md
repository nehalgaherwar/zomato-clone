# Zomato Clone

A simple clone of the Zomato food delivery platform built with HTML, CSS, and JavaScript.

## Features

- Search for restaurants
- View popular restaurants with images
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in your web browser to view the site.
